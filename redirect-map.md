1. **Priority 1**: Redirect all pages going to pillar pages first

### Netlify.toml Format

```toml
[[redirects]]
  from = "/old-url/"
  to = "/new-url/"
  status = 301
  force = true
```

### Verification Steps

After implementing redirects:
1. Run `npm run build` to ensure no broken internal links
2. Test redirects locally with `netlify dev`
3. Check Google Search Console for crawl errors after deployment
4. Update sitemap.xml to remove old URLs
5. Monitor 404 errors in analytics
