## ⚠️ Important: Existing netlify.toml Redirects

The `netlify.toml` file already contains ~100+ redirect rules that consolidate pages to hub pages (e.g., many commercial construction articles → `/commercial-construction/`).

**How redirects are processed:**
1. `_redirects` file takes precedence over `netlify.toml` redirects
2. Our new `_redirects` file maps placeholders to their **best matching canonical articles**
3. The `netlify.toml` consolidations may then redirect those articles to hub pages

**Example chain:**
```
/bacony-restoration.../ (_redirects) → /balcony-restoration-and-exterior-waterproofing-for-longevity/
                         (netlify.toml) → /balcony-reconstruction/
```

This is intentional - the placeholder gets the correct semantic redirect, and if that page is later consolidated, the chain still works.
